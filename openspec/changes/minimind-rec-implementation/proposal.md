## Why MiniMind-Rec

Adapt Mini-DeepLLM into recommendation semantic enhancement.
Generate item semantic tags and user interest summaries from text.

Code base: d:/my-projects/Mini-DeepLLM (~4400 lines).
Reference: github.com/jingyaogong/minimind

## What Changes
- Create item semantic tag SFT dataset from Amazon metadata
- LoRA fine-tune Mini-DeepLLM for tag generation
- Deploy trained model for offline inference on all items
- Evaluate tag quality impact on cold-start recommendation

## Capabilities
- data-prep: Amazon item text SFT dataset construction
- lora-finetune: LoRA fine-tuning for semantic tag generation
- tag-inference: Batch inference for all items
- cold-start-eval: Evaluate tag impact on cold-start recall

## Dataset
Amazon Beauty/Sports item metadata (title, category, description).
SFT format: input=title+category, output=semantic tags.
Tags generated by LLM API as pseudo-labels.
